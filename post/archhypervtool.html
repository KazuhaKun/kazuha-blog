Arch安装Hyper-v增强工具

发表于2022-01-22|更新于2022-01-26|虚拟机Linux
|字数总计:277|阅读时长:1分钟|阅读量:

不需要图形界面

  1. 按照正常步骤安装Arch并完成配置：
  2. 按照ArchWiki 中Post-installation下的第一部分，安装hyperv 并设置hv_fcopy_daemon.service hv_kvp_daemon.service  hv_vss_daemon.service的自启动 
    systemctl enable services.name //设置自启动，关闭则为disable
systemctl start services.name  //立即启动，暂停则为stop，重启为restart
    systemctl enable hv_fcopy_daemon.service
systemctl enable hv_kvp_daemon.service
systemctl enable hv_vss_daemon.service

需要图形界面且启动增强模式支持

  1. 按照不需要图形界面的步骤完成配置 以下按照官方wiki的Post-installation中Xorg后面的内容总结 Hyper-V
  2. 安装显卡驱动xf86-video-fbdev
  3. 安装增强功能（xrdp）：
git clone https://github.com/Microsoft/linux-vm-tools
cd linux-vm-tools/arch
./makepkg.sh
./install-config.sh
  1. 在宿主机的Powershell里输入以下代码以启动：
Set-VM -VMName 虚拟机名称 -EnhancedSessionTransportType HvSocket
  1. 编辑”~/.xinitrc” （没有的话手动创建） 添加如下代码以启动桌面： Xfce： 
    exec dbus-launch startxfce4
    KDE: 
    exec dbus-launch plasma-x11
  2. 启动后会出现增强会话调整分辨率的窗口，选择好后便会显示xrdp，输入用户名密码后即可登入桌面
